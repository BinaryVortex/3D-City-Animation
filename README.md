# 3D City Animation

A mesmerizing 3D city animation built with HTML, CSS, and JavaScript using Three.js. Experience a cyberpunk-inspired cityscape with moving cars, particle effects, and interactive controls.

## Demo

![Demo Screenshot](Screenshot%202024-12-08%20015049.png)

View the live demo: [https://binaryvortex.github.io/3D-City-Animation/](https://binaryvortex.github.io/3D-City-Animation/)

## Features

- **3D Cityscape**: Randomized buildings with varying heights and positions
- **Interactive Controls**: Mouse and touch controls to rotate and explore the city
- **Moving Cars**: Animated cars traversing the city streets
- **Particle Effects**: Smoke particles adding atmospheric depth
- **Lighting and Shadows**: Dynamic lighting with soft shadows for realism
- **Responsive Design**: Adapts to different screen sizes
- **Cyberpunk Theme**: "Back To The Red" aesthetic with red fog and neon elements

## Technologies Used

- **Three.js**: For 3D rendering and graphics
- **Bootstrap**: For responsive layout and styling
- **GSAP (TweenMax)**: For smooth animations
- **HTML5, CSS3, JavaScript**: Core web technologies

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/BinaryVortex/3D-City-Animation.git
   ```

2. Navigate to the project directory:
   ```bash
   cd 3D-City-Animation
   ```

3. Open `index.html` in your web browser to view the animation.

## Controls

- **Mouse Movement**: Rotate the city view
- **Touch (Mobile)**: Drag to rotate the city
